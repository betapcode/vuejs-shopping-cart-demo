# vuex-shopping-cart-demo

> Demo shopping cart with vue, vuex, vue-router project

## List packages 
- vue-router
- vuex
- accounting-js

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
